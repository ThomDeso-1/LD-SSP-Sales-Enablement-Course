# Vantage SSP Cloud Sales Enablement Course

GitHub Pages-ready portfolio site for a six-module cloud sales enablement course.

## Structure

- `index.html` — site landing page
- `modules/` — learner-facing modules
- `supporting/` — supporting portfolio artifacts
- `assets/` — reserved for future shared assets

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and the `/ (root)` folder.
6. Save and wait for the site URL to be generated.
